💡 About us

Mission

Create low-cost, infinitely adaptable assistive tech that makes everyday communication accessible for hearing-impaired and non-verbal people globally.

Vision

Become a first-choice platform for gesture-to-speech conversion, integrating IoT glove, AI camera, and mobile companion apps for social inclusion.

• AI gesture recognition (MediaPipe) with 95% accuracy.

• Arduino flex-sensor glove, real-time feedback, and Bluetooth connection.

• Raspberry Pi camera mode for no-glove usage leveraging neural inference.

• 3-mode output: speech, text on screen, remote control via mobile.

🗺️ Roadmap

✅

Phase 0: Research and Planning

We studied the communication barriers of sign language for 70 million deaf people worldwide. We surveyed the deaf community in Kazakhstan. We selected the hardware: Arduino Mega, flex sensors, and Raspberry Pi. We designed the system: a smart glove combined with an AI camera.

✅

Phase 1: Prototype Development

We created a glove with 5 flex sensors. Arduino Mega was used to read the sensor data. We trained a database of 20 gestures (Russian and English). We also added voice output.

✅

Phase 2: AI Camera Integration

We used Raspberry Pi 5 with a camera. MediaPipe was integrated to track 63 hand landmarks. This allowed gesture recognition without the glove. Both modes work together.

⚙️

Phase 3: Testing and Calibration

The task is to test the system with real users. It is necessary to increase accuracy to 95% or higher, calibrate the sensors for different hand sizes, add an OLED display, and optimize the system speed.

⚙️

Phase 4: Feature Expansion

The task is to add phrase recognition and improve the algorithm so it adapts to the user. It is also necessary to reduce latency and add quick access to frequently used gestures.
